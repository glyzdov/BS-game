# BS-game — Design Docs

Game design documents for Beer Station Pub. Code-level conventions live in the
repository root `CLAUDE.md`; this folder is for scenario, mini-game mechanics,
characters, quests, and economy.

## Structure

- `scenario.md` — high-level story and progression for both playable roles.
- `minigames/` — one document per mini-game, describing mechanic, inputs,
  failure/success states, and art/sound notes.

All documents are written in English. Keep them short, concrete, and updated
alongside the code they describe.
